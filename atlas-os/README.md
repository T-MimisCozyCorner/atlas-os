# ATLAS OS

ATLAS OS is an AI Business Operating System for creators, digital product sellers, affiliate marketers, and small business builders.

## Version
0.1.0 Alpha

## What is included
- Next.js app structure
- Tailwind CSS design system
- Dashboard
- Business Manager
- Product Manager
- Task Board
- AI Employees panel
- Asset Library placeholder
- Revenue Tracker

## Setup directions

### 1. Install Node.js
Download Node.js LTS from the official Node.js website.

### 2. Open this folder in VS Code

### 3. Install dependencies
```bash
npm install
```

### 4. Start development server
```bash
npm run dev
```

### 5. Open the app
Go to:

```bash
http://localhost:3000
```
